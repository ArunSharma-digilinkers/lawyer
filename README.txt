---------------------------------------------------
  3. INSTALLATION
---------------------------------------------------
1. Unzip the package.
2. Open index.html in your browser to preview.
3. To edit, open any .html file in a text editor (VS Code, Sublime).
4. CSS is inside assets/css/style.css.
   - If you use SCSS, edit files in assets/scss/ and recompile.

---------------------------------------------------
  4. CUSTOMIZATION
---------------------------------------------------
- Logo: Replace logo.png inside assets/img/ with your own.
- Colors: Update variables inside assets/scss/_variables.scss.
- Fonts: Change Google Fonts link in the <head> section of each HTML file.
- Contact Form: Edit assets/js/main.js with your email or configure PHP mailer.
